# Java Quiz Application

This project is a Java-based quiz software designed to allow users to take quizzes through a graphical interface. It demonstrates object-oriented programming concepts, quiz management, and automated testing using JUnit.

## Features

- GUI-based quiz system
- Question and quiz management
- Multiple-choice questions
- Score evaluation
- Modular Java design
- JUnit testing for reliability

## Technologies Used

- Java
- Object-Oriented Programming
- JUnit Testing Framework
- Hamcrest Library

## How It Works

The application loads quiz questions and allows users to answer them through a graphical interface. The system evaluates answers and calculates the final score.

JUnit test cases are included to verify the functionality of quiz and question modules.

## Learning Objectives

This project demonstrates:

- Java GUI programming
- Object-oriented design principles
- Software modularization
- Automated testing using JUnit
